Icon File Required
==================

For the shortcuts to work properly, you should place a "palmexitgarage.ico" file in this directory.

This icon will be used for:
- Desktop shortcuts
- Start Menu shortcuts
- Application branding

If no icon is provided, Windows will use the default .bat file icon.

Recommended icon specifications:
- File format: .ico
- Size: 32x32, 48x48, and 256x256 pixels
- File name: palmexitgarage.ico

You can create an icon using online tools or icon editors, or use any existing .ico file.
